## вңЁ Features

<details>
<summary><b>вқӨпёҸ Heart Disease Prediction</b> вҖ” click to expand</summary>

<br>

Predict **10-year cardiovascular risk** using a Gradient Boosting Classifier trained on clinical data.

**Input Features:**
| Feature | Type | Description |
|---|---|---|
| `age` | Numeric | Patient age in years |
| `sysBP` | Numeric | Systolic blood pressure |
| `glucose` | Numeric | Blood glucose level |
| `totChol` | Numeric | Total cholesterol |
| `BMI` | Numeric | Body mass index |
| `currentSmoker` | Binary | Smoking status |
| `cigsPerDay` | Numeric | Cigarettes per day |
| `BPMeds` | Binary | On BP medication |
| `prevalentHyp` | Binary | Hypertension history |
| `diabetes` | Binary | Diabetes history |

</details>

<details>
<summary><b>рҹ’ү Diabetes Detection</b> вҖ” click to expand</summary>

<br>

ML model trained on **100K+ clinical records** with top-tier accuracy.

**Key Features:**
| Feature | Importance | Description |
|---|---|---|
| `HbA1c_level` | рҹ”ҙ High | Hemoglobin A1c |
| `blood_glucose_level` | рҹ”ҙ High | Fasting glucose |
| `bmi` | рҹҹЎ Medium | Body mass index |
| `age` | рҹҹЎ Medium | Patient age |
| `smoking_history` | рҹҹў Low | Smoking status |
| `hypertension` | рҹҹў Low | BP condition |
| `heart_disease` | рҹҹў Low | Cardiac history |

</details>

<details>
<summary><b>рҹ§  Brain Tumor Detection</b> вҖ” click to expand</summary>

<br>

Deep-learning image classifier trained on MRI scans to detect:

- `Glioma`
- `Meningioma`
- `Pituitary`
- `No Tumor`

**Input:** MRI image (JPG/PNG) + patient basic information.  
**Output:** Predicted class, confidence score, risk level and downloadable PDF report.

</details>

<summary><b>рҹ“Ҡ Exploratory Data Analysis</b> вҖ” click to expand</summary>

- **Class Distribution** pie charts
- **BMI vs Age** scatter plots

## вҡҷпёҸ Setup & Run Locally

### Prerequisites

```bash
python --version   # Python 3.11+
pip --version      # pip 22+
```

### Step 1 вҖ” Clone

```bash
git clone https://github.com/BhavyaKansal20/healthy-ai.git
cd healthy-ai
```

### Step 2 вҖ” Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 3 вҖ” Train Models

```bash
python train_models.py
# вң” heart_model.pkl saved
# вң” diabetes_model.pkl saved
# вң” brain_tumor_model.pt saved
# вң” scalers and encoders exported
```

### Step 4 вҖ” Launch

```bash
python app.py
# рҹҡҖ Running on http://localhost:5000
```
