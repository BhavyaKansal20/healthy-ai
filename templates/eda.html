Mathematical Foundation

Gradient Boosting
F(x) = F₀ + Σₘ η·γₘ·hₘ(x)
Each iteration fits a weak learner (shallow tree) on the residuals of the previous model. n_estimators={% if model_type=='heart' %}200{% else %}150{% endif %}, max_depth=4.
Loss Function
Binary Cross-Entropy:
L = −[y·log(p) + (1−y)·log(1−p)]
Minimized via gradient descent each boosting step. Learning rate η = 0.1 prevents overfitting.
ROC-AUC Score
AUC = P(score_pos > score_neg)
1.0 = perfect classifier, 0.5 = random guess.
